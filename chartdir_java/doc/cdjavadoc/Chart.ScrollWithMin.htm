ChartDirector 6.1 (Java Edition)

Chart.ScrollWithMin

Declaration

public const int ScrollWithMin

Description

A constant equals to 3 to represent that the viewport should be adjusted so keep the distance from the axis end points to the lower full range unchanged.

This constant is used in ChartViewer.updateFullRangeH and ChartViewer.updateFullRangeV
